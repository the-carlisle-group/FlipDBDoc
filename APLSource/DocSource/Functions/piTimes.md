# piTimes

Type:{.prefix}

Scalar{.info}

Categories:{.prefix}

Arithmetic{.info}

~~~
R=piTimes X
~~~

X must be numeric. R is numeric.

The value of R is the product of the mathematical constant Pi times Y.

## Examples

~~~
      piTimes 0 0.5 1 2
┌──────────────────┐
↓0                 │
│1.5707963267948966│
│3.141592653589793 │
│6.283185307179586 │
└Float─────────────┘
~~~
