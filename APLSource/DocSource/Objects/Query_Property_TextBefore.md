# TextBefore Property

Applies to:{.prefix}

竊端##.##.Query]{.info}

The TextBefore property specifies a 竊端*.Objects.Text] object to be rendered before the query when
the query is embedded in a 竊端*.Objects.Report] It has no effect on the query itself, and is for
presentation purposes only. It is generally used to provide a title and additional introductory
