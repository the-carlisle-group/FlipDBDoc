# maximum

Type:{.prefix}

Scalar{.info}

Categories:{.prefix}

Arithmetic{.info}

higher, greater{.purpose}

~~~
R=X maximum Y
~~~

X and Y must both be numeric. R is the greater value of X and Y.

See also: →[max], →[minimum]

## Examples

~~~
      2 maximum 1 2 3
┌────┐
↓2   │
│2   │
│3   │
└Int8┘
~~~
