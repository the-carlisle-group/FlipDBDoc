# generatePSA

Type:{.prefix}

Structural{.info}

Categories:{.prefix}

Financial{.info}

Generate Public Securities Administration standard prepayment model.{.purpose}

~~~
R=[Y] generatePSA X
~~~

X is a percentage value. Y is optional loan age. R is age-adjusted X percent of PSA.
