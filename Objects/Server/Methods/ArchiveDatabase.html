The ArchiveDatabase method archives a database by first making a full, backup copy of the existing database and then removing all inactive rows from the existing database. FlipDB employs an automatic archive naming and numbering system.

For example:

              S.ArchiveDatabase 'SANDP'

will create a new database SANDP_ARCH_001, and remove all inactive rows from SANDP.
