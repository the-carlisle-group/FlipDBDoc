CopyDatabase Method

Applies to:{.prefix}

→[##.##.Server]{.info}

This method makes a copy of a database.

The argument is composed of 2 or 3 items:

- -
2 Target database name String
[3] Purge Boolean

The result is composed of 1 item:

- -

The optional third argument specifies whether or not the target tables are purged of deleted and inactive rows. If the Purge parameter is 0, the default, the target database is an exact copy of the source, and includes all historical data and transaction ID's. If the Purge parameter is 1, then only active rows are copied into the target database, and this database will have no historical data. See the →[*.Purge] method of the Table object for a discussion of historical data.

For example:

              D=S.CopyDatabase 'SANDP' 'SANDP_COPY'
