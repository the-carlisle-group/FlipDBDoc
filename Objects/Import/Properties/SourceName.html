SourceName Property

Applies to:{.prefix}

→[##.##.Import]{.info}

If the SourceType property is "ADO", this property is provided as the "Data Source" parameter of the connection string.

If the SourceType property is "ODBC", this property is provided as the DSN (Data Source Name) parameter of the connection string.

In both ADO and ODBC, this property may be left empty and the associated parameter specified directly in the →[ConnectionString] property.
