dropChar

Applies to:{.prefix}

Character{.info}

Type:{.prefix}

Scalar{.info}

Categories:{.prefix}

Text{.info}

Truncate strings.{.purpose}

        R=X dropChar Y

Y is Char. X is a scalar integer.

R is Char and is equal to Y with the first X characters removed if X is positive, or the last X characters removed if X is negative.

See also: →[takeChar]

Examples
