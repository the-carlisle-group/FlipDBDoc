roundByFactor

Type:{.prefix}

Scalar{.info}

Categories:{.prefix}

Arithmetic{.info}

        R=[X] roundByFactor Y

X and Y must be numeric. R is Y rounded to the nearest multiple of X. The default value for X is 1.

See also: →[round] →[bankersRound]

Examples
