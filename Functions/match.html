match

Type:{.prefix}

Structural{.info}

Categories:{.prefix}

Relational{.info}

equivalent to{.purpose}

        R=X match Y

X and Y are columns of any type. R is a boolean scalar, being 1 if the value of X is exactly the same as that of Y; 0 otherwise.

See also: →[eq]

Examples

              1 2 3 match 1 2 3
┌───────┐
│1      │
└Boolean┘
      1 2 match 'hello'
┌───────┐
│0      │
└Boolean┘
      1 match enlist 1
┌───────┐
│0      │
└Boolean┘
