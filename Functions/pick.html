pick

Type:{.prefix}

Structural{.info}

Categories:{.prefix}

Selection{.info}

Select an item by index.{.purpose}

        R=X pick Y

X is zero, one or two →integers. Y is any column or object array. R is one of:

element of Y yielding a scalar * A disclosed partition of Y yielding a simple column * An object of Y * All

Zero (0) picks the first or only item of a structure with subsequent items being picked by increasingly higher integers. If X is empty, R is all of Y. If X is a single number (n), R is the nth element of simple Y, the disclosed contents of the nth partition of partitioned Y or the nth object of array Y. If X is two numbers (m,n), they pick the nth item of the mth partition of partitioned Y.

See also: →[index]

Examples
