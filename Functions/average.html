average

Type:{.prefix}

Aggregate{.info}

Categories:{.prefix}

Statistical,Date/Time{.info}

Calculates the average value{.purpose}

        R=average X

X must be numeric or temporal. If X is numeric, R Float. If X is temporal, then R is the same type as X.

This function is identical to →[mean].

See also: →[weightedAverage]

Examples
